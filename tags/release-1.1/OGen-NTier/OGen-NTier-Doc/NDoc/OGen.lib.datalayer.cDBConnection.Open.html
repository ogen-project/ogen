cDBConnection.Open Method 

Opens DataBase Connection.

public void Open();

Remarks

There is no need to Open a Connection in order to execute an SQL Query or Function. Open and Close methods are usually used when working with the Transaction

Exceptions

Exception Type Condition
InvalidConnectionstringException_empty Thrown when an empty Connection String has been supplied
OpenException_alreadyOpened Thrown when the Connection is already opened
