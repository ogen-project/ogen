If you don't plan to build from source or contribute to OGen you'll only need:
  • Npgsql
  • MySQL Connector/Net (support is on the way)
If you're building from source, or plan to contribute you'll also need:
  • build source files using SharpDevelop or Visual C# 2005 Express Edition (NAnt and MonoDevelop not supported yet)
  • a subversion client like TortoiseSVN (for windows) or Subversion (multi-platform)
  • NUnit
  • NDoc
