2.2. installing ogen

dependencies and pre-requisites

If you don't plan to build from source or contribute to OGen you'll only need:
  • Npgsql
  • MySQL Connector/Net (support is on the way)
If you're building from source, or plan to contribute you'll also need:
  • build source files using SharpDevelop or Visual C# 2005 Express Edition (NAnt and MonoDevelop not supported yet)
  • a subversion client like TortoiseSVN (for windows) or Subversion (multi-platform)
  • NUnit
  • NDoc


installing OGen under windows

OGen at BerliOS: http://developer.berlios.de/projects/ogen /
  1. Building from Source:

    1. Anonymous Subversion Access via SVN: svn://svn.berlios.de/ogen/trunk/
      or HTTP: http://svn.berlios.de/svnroot/repos/ogen/trunk/

    2. Once you have the sources, check the 'distro' directory for solution and scripts files.

      1. win-100-generate-keys.bat
        generates keys needed to compile assemblies with strong name so they can be added to the GAC

      2. win-110-generate-app.config.bat
        generates app.config with your local path to OGen project

      3. win-120-buildAssembies-7.1-registry-tweak4visualstudio.js
        if you're using Visual Studio 2003, you'll need to define where to find dependant asseblies (even if they're in the GAC)

      4. win-120-buildAssembies-7.1.sln (Visual Studio 2003)
        win-120-buildAssembies-8.sln (SharpDevelop, Visual C# 2005 Express Edition or Visual Studio 2005)
        win-120-buildAssembies.build (NAnt, not yet supported)

        you should set build configuration to RELEASE and compile using any of the provided solutions

      5. win-130-copyAssemblies-1.1.bat
        copies (Visual Studio 2003) compiled assemblies (framework 1.1) to bin directory

      6. win-130-copyAssemblies-2.0.bat
        copies (Sharp Development / Visual Studio 2005) compiled assemblies (framework 2.0) to bin directory

      7. win-140-installAssemblies-intoGAC-1.1.bat
        adds (Visual Studio 2003) compiled assemblies (framework 1.1) to the GAC

      8. win-140-installAssemblies-intoGAC-2.0.bat
        adds (Sharp Development / Visual Studio 2005) compiled assemblies (framework 2.0) to the GAC

      9. win-150-OGen.ALL.Templates-install-1.1.bat
        copies (Visual Studio 2003) compiled assembly (framework 1.1) for the OGen-NTier, OGen-Doc and OGen-XSD Templates to bin directory and generates web.config file

      10. win-150-OGen.ALL.Templates-install-2.0.bat
        copies (Sharp Development / Visual Studio 2005) compiled assembly (framework 2.0) for the OGen-Doc, OGen-Doc and OGen-XSD Templates to bin directory and generates web.config file

    3. Once you have the compiled assemblies installed in the GAC, check the 'distro\bin' directory where you'll find OGen binaries and generation templates.
