the basics
the four basic operations over a db table

Insert / insObject

Assigning Fields for Table User, Inserting and retrieving Sequence for IDUser Field:
DO_User _user;
long IDUser = 0L;

_user = new DO_User();
_user.Login = "fmonteiro";
_user.Password = "password";
_user.insObject(true); // assigning selectIdentity parameter to true retrieves Sequence for IDUser
Console.WriteLine(
  "user inserted, at:\nIDUser = {0}", 
  IDUser = _user.IDUser
);
_user.Dispose(); _user = null;

Select / getObject

Reading Table User Fields, based on IDUser Field:
_user = new DO_User();
if (
  _user.getObject(IDUser) // based on it's PK
) {
  Console.WriteLine(
    "successully read Fields for User: {0}\n\tlogin: {1}", 
    _user.IDUser, 
    _user.Login
  );
} else {
  Console.WriteLine(
    "couldn't read Fields for User: {0}"
    _user.IDUser
  );
}

Update / updObject

Updating Table User Login Field:
//_user.getObject(IDUser); // comment, no need if you've done it already
_user.Login = "francisco";
_user.updObject(false); // assigning forceUpdate parameter to false Updates only if changes have been made

Delete / delObject

Deleting User from Table, based on IDUser Field:
_user.delObject(IDUser); // based on it's PK
_user.Dispose(); _user = null;
