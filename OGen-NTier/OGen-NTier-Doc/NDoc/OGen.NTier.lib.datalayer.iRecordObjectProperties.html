iRecordObject Properties

The properties of the iRecordObject interface are listed below. For a complete list of iRecordObject interface members, see the iRecordObject Members topic.

Public Instance Properties

Count Represents number of items in the Record.
Current Represents current iteration at the Record.
Fullmode Indicates Record mode, True if in Fullmode, False if not.
isOpened Indicates if Record is open, True if is open, False if not.
Record Exposes Record DataTable.
