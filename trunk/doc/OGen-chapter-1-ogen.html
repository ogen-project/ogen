1. OGen - open generation

1.1. introduction

What is OGen

OGen is a code generator framework.

TODOS

It generates code on text files and databases(1) based on sets of XML metadatas(2) and aspx/XSLT templates provided by users.

(1) PostgreSQL, SQL Server and MySQL are currently supported
(2) gathered from SQL information schema, user specific XML data and .net assemblies

 Audience

Then main audience for the OGen code generator framework are developers
