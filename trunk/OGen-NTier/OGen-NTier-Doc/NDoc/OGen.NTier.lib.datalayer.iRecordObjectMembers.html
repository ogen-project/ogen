iRecordObject Members

iRecordObject overview

Public Instance Properties

Count Represents number of items in the Record.
Current Represents current iteration at the Record.
Fullmode Indicates Record mode, True if in Fullmode, False if not.
isOpened Indicates if Record is open, True if is open, False if not.
Record Exposes Record DataTable.

Public Instance Methods

Close Closes Record.
EOR Indicates if current iteration is at the End Of Record.
Open Opens Record, based on some DataTable.
ReadOverloaded. Reads values from Record, assigns them to the appropriate DataObject property, finally it steps current iteration at the Record forward and returns a bool value indicating if End Of Record has been reached.
Reset Points current iteration to the Beginning of the Record.
